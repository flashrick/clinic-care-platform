# clinic-care-platform

A full-stack web platform for small clinics to manage patient appointments and prescriptions.

## Summary
Patients can register, book appointments (planned voice-recognition notes component), and access treatment history.  
Doctors can view schedules, write prescriptions, and manage patient profiles.

## Tech Stack
- Backend: Python 3.10+, Django, Django REST Framework (DRF)
- Frontend: Vite + React + TypeScript, Tailwind CSS, daisyUI
- Database: PostgreSQL
- Auth: JWT (djangorestframework-simplejwt)
- API Docs: drf-yasg (Swagger UI)
- Testing: PyTest
