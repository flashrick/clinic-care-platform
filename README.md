# clinic-care-platform

A full-stack web platform for small clinics to manage patient appointments and prescriptions. :contentReference[oaicite:0]{index=0}

## Summary
Patients can register, book appointments (including a planned voice-recognition notes component), and access treatment history. Doctors can view schedules, write prescriptions, and manage patient profiles. :contentReference[oaicite:1]{index=1}

## Tech Stack
- Backend: Python 3.10+, Django, Django REST Framework (DRF) :contentReference[oaicite:2]{index=2}
- Frontend: Vite + React + TypeScript, Tailwind CSS, daisyUI :contentReference[oaicite:3]{index=3}
- Database: PostgreSQL :contentReference[oaicite:4]{index=4}
- Auth: JWT (djangorestframework-simplejwt) :contentReference[oaicite:5]{index=5}
- API Docs: drf-yasg (Swagger UI) :contentReference[oaicite:6]{index=6}
- Testing: PyTest :contentReference[oaicite:7]{index=7}
